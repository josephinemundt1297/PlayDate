# PlayDate – Dokumentation

Hier liegen alle ausführlichen Projektunterlagen an einem Ort.

| Dokument | Inhalt |
| --- | --- |
| [`konzept.md`](konzept.md) | Produktidee, Zielgruppe, MVP und Leitplanken |
| [`speck.md`](speck.md) | Anforderungen, Datenmodell und Abnahmekriterien |
| [`phasenPlan.md`](phasenPlan.md) | Phasen, Checklisten, Risiken und Auditplan |

`README.md` und `AGENTS.md` bleiben im Repository-Stamm. GitHub zeigt die README dort automatisch an, und Entwicklungswerkzeuge finden dort die Arbeitsregeln.

PlayDate ist ein Übungsprojekt. Die Unterlagen simulieren eine saubere Produktentwicklung, sind aber keine Rechtsberatung und keine Freigabe für den Betrieb mit echten personenbezogenen Daten.
